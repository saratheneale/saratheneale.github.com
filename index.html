Welcome to GitHub Pages.

My name is Sara, and I'm a coder. I've been a little low on code lately, and I'm diving back into it by learning some JavaScript. I'll share with you!

Here are a few JavaScript projects:

ChatBots IRC: These are Chrome applications that utilize the Chrome Sockets API. To get them to work, you'll have to load them as unpacked extensions and generate a background page.

  • OptimistBot- OptimistBot loves encouraging channel conversation. He says positive things when you mention him.
  • GoodbyeBot- says goodbye and leaves the channel when someone says hello. GoodbyeBot was recently refactored to include an IRCClient Class. I have lots of bot ideas and don't want to copy paste.
  • IRCClient - a class/library for basic IRC connecting using the Chrome Sockets API

littleNoAd- a tiny ad blocker for my personal use.

LolCat Memory - Classic Memory tile game with randomly generated LolCat images. This was a Girl Develop It pair up programming exercise. My version navigates the DOM with plain JavaScript. I'll convert it to jQuery later for fun.

[Loljban Memory]- Link Coming soon after some revision. Lojban is a constructed language based on predicate logic with a small but dedicated community learning and contributing to the language. A friend of mine is learning it. He saw LolCat Memory and wanted me to combine it Lojban words with images of cats related to the words. Hence Lol-jban Memory was born.
